SK-Gaming arena ranking
If you've been reading this site for a (very) long time, you might remember the old arena ranking from geekboys.org posted a while ago. It's been moved to SK-Gaming.com (http://www.sk-gaming.com/arena/all/all/all/all/all/) with some new features included. Now you can :
See the top arena teams for each bracket (5v5, 3v3, 2v2) sorted by Region/Battlegroup/Server
Check the top players for each class on each bracket (e.g. best hunter playing in 5v5 bracket), sortable by Region/Battlegroup/Server as well.
That's the new feature, you can now see the player ranking for each class based on their score in all brackets. For the moment, the player with the highest combined rating (http://www.sk-gaming.com/arena/player/) in the world is Cherekee (http://eu.wowarmory.com/character-sheet.xml?r=Talnivarr&n=Cherekee), a hunter on EU-Talnivarr.



Note : The tools have been updated recently, and the armory had a few major downtimes during the last few weeks, you might experience a few problems with some data not being updated correctly but most of the time it should work without any problem.
