chaud2015-02-23, 12:11 PMPatch 6.1 Live This Week
Patch 6.1 goes live this week, bringing us some systems changes and content. We will be updating this post as we wait for the patch to go live, so keep checking back!


New Blood Elf Models - The updated models are finally here!
New Character Model Tweaks - The Night Elf male run animation was finally changed.
Heirloom Tab - The Heirloom tab is finally here!
Heirloom Item Changes and Upgrades - Item upgrades are back for Heirlooms.
New Neck Heirlooms - The new neck slot heirlooms provide a heal on a 2 minute cooldown.
Heirloom Achievements and Mount - A new 60% mount usable from Level 1 and some achievements are part of the heirlooms update.
Heirloom Prices - Heirloom prices have been adjusted and some new heirlooms added.
Twitter Integration - Link a Twitter account and tweet selfies.
S.E.L.F.I.E - The most useful garrison mission reward: A selfie camera that can be upgraded to add filters.
Follower Changes - Harrison Jones, a new legendary follower, and more!
Mission Changes - Profession missions, Archaeology missions, BRF missions, and Relic Hunter missions are just some of the new missions added.
Garrison Resource Changes - Scavenger http://ptr.wowdb.com/garrison/abilities/79-scavenger was nerfed, new items added to the vendor, and a new source of garrison resources were added.
Work Order / Building Changes - Faster work order completion, more useful buildings, and other quality of life changes.
Garrison UI Changes - The default UI has a few basic Master Plan http://www.curse.com/addons/wow/master-plan features now.
Garrison Jukebox - The ability to change your garrison's music and another collection minigame all in one.
Misc Garrison Changes - Friends now have a reason to visit your garrison, including daily quest givers, platinum invasions, and raid bosses.
New Apexis Gear - If you have the crystals, the vendors have some item level 670 armor for you!
Profession Changes - Stage 4 crafted items, crafting spells that bypass the daily cooldown, and a Savage Blood transmute http://ptr.wowdb.com/spells/181643-transmute-savage-blood are a few of the changes.
Graphical Updates - Better SSAO, MSAA is back, SSAA has been added, and nicer lighting.
New Loading Screens - Garrisons finally get their own loading screens.
Veteran Accounts - Your characters under level 20 can log in and join your guild even without an active subscription. 
New Toys - Along with the new toys, several items were flagged as toys so that they can be added to the Toy Box.
Mystic Runesaber Mount - The Mystic Runesaber http://ptr.wowdb.com/spells/180545-mystic-runesaber ground mount will be purchased from the in-game shop.
Colorblind Support - Differentiating colors should be a little easier for those with colorblindness now.
Addons - As always, we need your help collecting data from in game by installing the WoWDB addon.

Heirloom Tabheirloomtab
Patch 6.1 adds the long awaited Heirlooms Tab!


The heirloom collection is sorted by slot and can be filtered by class and spec. 
Existing heirlooms are added to the collection as soon as you log in to the character with them. 
Most of your existing heirlooms will be added with a 1/2 upgrade level that scales them to Level 90. You will keep the existing heirloom item, so don't worry about losing existing enchants.
Garrosh heirlooms don't show in the collection unless you already own them. If you own one, the most powerful version of it will show up in your collection, scaling to Level 100.
Clicking on a heirloom in your collection will create a new copy of the heirloom. This copy can be enchanted like any other item, but it will only affect that item. If you want to use the enchanted item on another character, you will need to mail it to the other character.
Bind on Account items from Archaeology are not added to this collection. 
Heirlooms with PvP Power or Resilience have had those stats converted into other, non-PvP secondary stats.
Stats have been removed, added, or adjusted on many heirloom items.

Heirloom Item Changes and UpgradesheirloomChanges
Patch 6.1 makes a few changes to how heirlooms work and how they are obtained. 


The heirloom vendors that take gold are Estelle Gendry http://ptr.wowdb.com/npcs/6566-estelle-gendry in Undercity and Krom Stoutarm http://ptr.wowdb.com/npcs/6294-krom-stoutarm in Ironforge.
These NPCs sell heirlooms that scale to Level 60 http://ptr.wowdb.com/items?filter-patch=19445&filter-quality=128. 
You can purchase items to upgrade your heirloom to scale to Level 90 and then Level 100.
There is a quest chain which allows you to earn some upgrades:

A breadcrumb quest http://ptr.wowdb.com/quests/38306-mystery-notebook sends you to your faction's heirloom vendor, who gives a quest to collect Misprinted Draenic Coin http://ptr.wowdb.com/items/122618-misprinted-draenic-coin.
Coins are found in Bounty Payouts, Relic Acquisition Compensatory Packages, and Garrison Resource Shipments.
The first quest http://ptr.wowdb.com/quests/38345-numismatics rewards a Ancient Heirloom Armor Casing http://ptr.wowdb.com/items/122338-ancient-heirloom-armor-casing for 5 x Misprinted Draenic Coin http://ptr.wowdb.com/items/122618-misprinted-draenic-coin.
The next quest http://ptr.wowdb.com/quests/38394-completionism rewards Timeworn Heirloom Armor Casing http://ptr.wowdb.com/items/122340-timeworn-heirloom-armor-casing for 10 x Misprinted Draenic Coin http://ptr.wowdb.com/items/122618-misprinted-draenic-coin.
The third quest rewards Ancient Heirloom Scabbard http://ptr.wowdb.com/items/122339-ancient-heirloom-scabbard for 25 x Misprinted Draenic Coin http://ptr.wowdb.com/items/122618-misprinted-draenic-coin.
A fourth quest hasn't show up yet, but if it exists, it may reward Timeworn Heirloom Scabbard http://ptr.wowdb.com/items/122341-timeworn-heirloom-scabbard for an increased coin cost.

You can still purchase heirlooms for Honor, Darkmoon Faire Tickets, Argent Tournament seals, and many holiday currencies. There is a nice listing at the bottom of this post.

WoWDB Addon Data Collectionwowdbaddons
Quickly populating the database with 6.1 additions is done with the help of many players. The more people that contribute data, the more accurate and quickly we can update the database. 

To help out, just click here curse://enable-wowdb-upload if you have Curse Client already installed. 

When you are playing the addon will gather data and save it. When you are done playing, Curse Client will see the game close and upload the collected data from the addon. You can see the last time data was uploaded by looking in the Plugins tab of the options, as well as making sure the addon is enabled there. Be sure you have Curse Client open before you close WoW so that the data will upload! 

The addon now supports Live, Beta, and the PTR, so be sure to add all of your WoW installs to the client.
