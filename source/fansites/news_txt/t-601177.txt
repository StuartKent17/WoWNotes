Boubouille2007-04-04, 08:27 PMMicrosoft released yesterday a hotfix for Windows 98, 2000, 2003 , XP , and Vista.

This was released to fix the "Windows Animated Cursor Remote Code Execution Vulnerability".

This vulnerability has to be fixed for WoW players because it is used to install a new version of an already well-known password stealer.
