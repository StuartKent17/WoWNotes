Legion - Account Wide Challenge Mode Weapon Transmog
The long asked question about transmogging Challenge Mode weapons has finally been answered!
Will players that completed WOD challenge modes unlock all the weapon models for xmog? If not seems unfair to some classes.
In #Legion you will be able to purchase a token that will unlock all Challenge Mode weapon appearances account-wide!

To clarify, only players who have completed all the Warlords of Draenor Challenge Mode Golds will be able to buy this token. (Blue Tracker (http://blue.mmo-champion.com/topic/399258-draenor-cm-weapons-legion-transmog/) / Official Forums (http://eu.battle.net/wow/en/forum/topic/17288700635?page=1#11))

Yep! Glad to hear this has gotten around, as we haven't been ignoring this topic at all. In fact, this is something that we've been discussing for quite some time. We wanted to be certain that we could completely decided on the plans for these weapons before we said anything.

We'd rather not make promises we aren't completely certain we can keep. (Blue Tracker (http://blue.mmo-champion.com/topic/399234-wod-cm-weapon-transmogging-in-legion-resolved/) / Official Forums (http://us.battle.net/wow/en/forum/topic/20419913969?page=2#30))
