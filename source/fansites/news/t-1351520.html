Patch 5.4 Raid Finder and Flex Drop Rates
Now that we have a few weeks worth of data on WoWDB (http://www.wowdb.com), we can talk a little bit about the Raid Finder and Flex difficulty drop rates.

World Boss Drop Rates

Celestials (http://www.mmo-champion.com/content/306-The-August-Celestials-Pandaria) - 21% item drop rate. The Celestial Treasure Box (http://www.wowdb.com/items/104272-celestial-treasure-box) drops 79% of the time.

Nalak (http://www.wowdb.com/npcs/69099-nalak) - 18% item drop rate. The Stormtouched Cache (http://www.wowdb.com/items/95602-stormtouched-cache) dropped 82% of the time.

Ordos (http://www.mmo-champion.com/content/807) - 15% item drop rate. The Flame-Scarred Cache of Offerings (http://www.wowdb.com/items/104273-flame-scarred-cache-of-offerings) dropped 85% of the time.

Oondasta (http://www.wowdb.com/npcs/69161-oondasta) - 19% drop rate. The Shiny Pile of Refuse (http://www.wowdb.com/items/95601-shiny-pile-of-refuse) dropped 81% of the time.
Galleon (http://www.wowdb.com/npcs/62346-galleon) - 17% item drop rate. The Marauder's Gleaming Sack of Gold (http://www.wowdb.com/items/90840-marauders-gleaming-sack-of-gold) dropped 83% of the time.



Raid Boss Drop Rates

Siege of Orgrimmar Raid Finder - 25% item drop rate. The Twisted Treasures of the Vale (http://www.wowdb.com/items/104275-twisted-treasures-of-the-vale) and Coalesced Turmoil (http://www.wowdb.com/items/104271-coalesced-turmoil) drop 75% of the time.

Throne of Thunder Raid Finder (Pre Patch 5.4) - ~16% item drop rate. The Treasures of the Thunder King (http://www.wowdb.com/items/95343-treasures-of-the-thunder-king) dropped ~84% of the time.

Siege of Orgrimmar Flex - 25% item drop rate. The Twisted Treasures of the Vale (http://www.wowdb.com/items/105713-twisted-treasures-of-the-vale) and Coalesced Turmoil (http://www.wowdb.com/items/105714-coalesced-turmoil) drop 75% of the time.



Siege of Orgrimmar Heroic Progression
Method (http://www.wowprogress.com/guild/eu/twisting-nether/Method/rating.tier16_25) and Blood Legion (http://www.wowprogress.com/guild/us/illidan/Blood+Legion/rating.tier16_25) are leading progress with 13/14 bosses down now, followed by six other guilds at 12/14. Over in the 10 player column, Paragon (http://www.wowprogress.com/guild/eu/lightning-s-blade/Paragon/rating.tier16_10) is still leading with 13/14 bosses down.

Upcoming Pet Changes
We have a few hotfixes to share with you and these changes should be in game now, or will take effect once the realms have been restarted:


Pierre (http://www.wowdb.com/npcs/70082-pierre) and Rascal-Bot (http://www.wowdb.com/npcs/71693-rascal-bot) are no longer Bind on Pickup; they can now be caged and you can collect up to 3 of each.
Rascal-Bot's Phaser spell is now a one-round ability. The tooltip still erroneously states that it is a two-round ability which we plan to correct at a later date.
Jadefire Spirit's Fade ability now has an eight-round cooldown.

Patch 5.4 and Deterrence
I understand why there would be some confusion about Deterrence's function in 5.4. And yes, it's a nerf in cases where you were using it more or less on cooldown every minute for longer than about 4 minutes.

However, in cases where you'd like to use it more than once inside a 1-minute period, the new Deterrence is a significant buff. Previously, that couldn't be done without using Readiness, and the vast majority of players weren't saving Readiness for Deterrence.

It's different now, sure, but we like how the new version is working. (Blue Tracker (http://blue.mmo-champion.com/topic/272320-im-calling-you-out-on-this-mess-up-fix-it/) / Official Forums (http://us.battle.net/wow/en/forum/topic/10104710390?page=4#71))

WoWDB Addon Data Collection
Collecting data on things like the 5.4 drop rates is done with the help of thousands of players. The more people that contribute data, the more accurate and quickly we can update the database.

To help out, just click here (curse://enable-wowdb-upload) if you have Curse Client already installed.

When you are playing the addon will gather data and save it. When you are done playing, Curse Client will see the game close and upload the collected data from the addon. You can see the last time data was uploaded by looking in the Plugins tab of the options, as well as making sure the addon is enabled there. If you don't normally leave it open, be sure you open Curse Client once in a while so that the data will upload
