In GMail, click the small downward arrow next to Reply, and click "Show Original". This will include the header information of the mail. Amoung the headers you should find some Received: lines, you can use those to check the mail servers the mail has been through. If it's legit it should start from a blizzard mail server.

For instance a line like this is a good sign:
Received: from eu-smtp01.blizzard.com (eu-smtp01.blizzard.com [80.239.174.216])
* * * * by mx.google.com with ESMTP id XXXXXX
