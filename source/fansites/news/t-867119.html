Patch 4.1: Addon Messages Will Be Filtered
Boring stuff you will probably not understand unless you're an addon developer, but it seems important!
Addon authors that use chat messages should check that their addons still work in 4.1

Authors will need to add calls to RegisterAddonMessagePrefix() in order to receive addon messages of a particular prefix, presumably during PLAYER_ENTERING_WORLD for the majority of addons. Note that this system errs on the side of sending too much; it is possible for addon messages to make it to the CHAT_MSG_ADDON handlers when no addon has registered it.



RegisterAddonMessagePrefix( &lt;string&gt; )

Returns a boolean if successful (could fail if it hits the client side limit of 512 prefixes).



IsAddonMessagePrefixRegistered( &lt;string&gt; )

Returns a Boolean if the prefix has been registered.



&lt;table&gt; GetRegisteredAddonMessagePrefixes( &lt;table&gt; )

Returns a table (or you can pass in a table), that will be filled in with all prefixes that were registered.



Prefixes are limited to 16 characters now, and they no longer take up a portion of the 256 character chat message limit. The server has a 64 prefix limit. If you exceed this limit your client will not filter any messages.

Also: SendAddonMessage() is now allowed for the OFFICER channel.

Quest Phasing
This is the sort of philosophy we're going for with phasing, as we do feel it's an important tool for storytelling. We do recognize issues where we've "over-phased" zones in the past though, and we generally tried to take a more minimal approach to phasing in Cataclysm. While there is still a good deal of phasing involved in many of the zones, we tried to make each phase a little more intricate and less intrusive on group gameplay. We want to make sure that, in times when a group is required, phasing doesn't hold you back from grouping with friends. (Source (http://blue.mmo-champion.com/topic/163327/phasing-a-huge-mistake))

Phased Ore/Herbs
It is an issue we're aware of and having been looking into. We've applied a number of hotfixes to phased gathering nodes since the release of Cataclysm, but this one is still there and is certainly a bugger. (Source (http://blue.mmo-champion.com/topic/163327/phasing-a-huge-mistake))

The addon changes are nice. What it means for users is:

- Slower/lower bandwidth clients will no longer be bogged down by addon message spam that they're receiving even if they don't use the addon. This is especially nice since every addon that looks for messages has to handle all of said incoming messages, even if it's just long enough to determine "Nope, not for me."

- Certain addons like EPGP Lootmaster that are intended to keep details between officers can now use the officer channel to share data rather than sending the data via raid or guild channels. A lot of loot systems by design don't want raiders to know what other raiders are expressing interest on an item (or how much DKP they're bidding/etc) but still need officers to know for loot council/etc decisions.

The bad news is that all addons that do talk/listen are going to be broken until they're patched -- though it sounds like it'd be trivial to write an addon that effectively undoes the change for short-term compatibility while you wait for updates.
