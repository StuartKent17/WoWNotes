Legion - Wardrobe UI
The new Wardrobe tab in Legion will help with your transmog sets!


Select an item slot from the top menu, including enchants.
Then scroll through all of the available appearances for that slot.
Hovering over an appearance tells you the item name, source, and other items that use the same appearance.
The top bar tells you how many of the available appearances you have unlocked.
You can search by name and use various filters.
You can save each set and link them to other players.
