Patch 3.3 Quests Changes - Cairne and Baine Bloodhoof

You might remember from the World of Warcraft - Cataclysm (http://www.mmo-champion.com/news-2/world-of-warcraft-cataclysm/) leak that Cairne would eventually .... "leave" Thunder Bluff. A new NPC has been added to the game - Baine Bloodhoof (Leader) (http://db.mmo-champion.com/c/36648/baine-bloodhoof-leader/).
A new version of Rites of the Earthmother (http://db.mmo-champion.com/q/776/rites-of-the-earthmother/) quest has been added to the game > Rites of the Earthmother (http://db.mmo-champion.com/q/14440/rites-of-the-earthmother/). Baine Bloodhoof replaces his father in this quest.

This change isn't active on PTRs for the moment but Blizzard is definitely getting a few things ready for when it will happen ...

I will also say that while I have read a large number of rumors and speculation concerning what's to come, I haven't seen a single one that has the progresson of events or the reasons behind them correct. ;) (Source (http://blue.mmo-champion.com/29/20140809141-why-garrosh.html))

Only one new class in 5 years?

Beyond the balance concern, which is a valid one, we don't want to over-saturate World of Warcraft with different classes. We want to make sure each class has a very distinct set of core mechanics from other classes. Continually introducing new classes can tend to bring about a feeling of redundancy if we're not fully comfortable with developing an entirely new set of mechanics distinct from the current classes.

This does not mean that we'll never consider adding new classes to the game as time moves on, but we're pretty happy with the play styles represented by the classes currently available. (Source (http://blue.mmo-champion.com/1/20140811526-1-new-class-in-5-years--not-acceptable.html))

Thanks and sorry for the mixup. (Source (http://blue.mmo-champion.com/1/20136595097-did-they-lower-riding-skill-costs-again.html))
