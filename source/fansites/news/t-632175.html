Update - Patch Notes (http://blue.mmo-champion.com/1/14697489930-patch-308a-notes.html) now available. Also added a few mages undocumented changes from yesterday.

Patch 3.0.8a on live servers
Patch 3.0.8a is being deployed on live servers, for the moment it looks like it's only here to fix most of the bugs mentionned earlier and a few other things.

http://static.mmo-champion.com/mmoc/images/icons/images.gif Mage (Skills List (http://www.mmo-champion.com/index.php?page=670) / Talent + Glyph Calc. (http://talent.mmo-champion.com/?mage))
Arcane Spec changes
It looks like the mage was hotfixed yesterday to make a few changes to the Arcane tree
The Arcane Blast buff is now instantly consumed by Arcane Missile to prevent people from clipping their cast to get a boosted Arcane Barrage too.
The Glyph of Arcane Blast (http://www.wowhead.com/?item=44955) now gives 3% additional damage and doesn't work on Arcane Missiles anymore.

http://static.mmo-champion.com/mmoc/images/icons/ideathknights.gif Death Knight (Skills List (http://www.mmo-champion.com/index.php?page=650) / Talent + Glyph Calc. (http://talent.mmo-champion.com/?deathknight))
Howling Blast 5 seconds cooldown not displayed on tooltip
Assuming nothing comes up at the last minute, this should be fixed tomorrow (Tue, Jan 27). (Source (http://blue.mmo-champion.com/28/14498471268-howling-blast-status.html))

http://static.mmo-champion.com/mmoc/images/icons/ihunters.gif Hunter (Skills List (http://www.mmo-champion.com/index.php?page=620) / Talent + Glyph Calc. (http://talent.mmo-champion.com/?hunter))
Hunter Aspects still behaving as if they still have a GCD
Assuming nothing comes up at the last minute, this should be fixed tomorrow (Tue, Jan 27). (Source (http://blue.mmo-champion.com/28/14498469718-hunter-aspects-and-the-gcd.html))

http://static.mmo-champion.com/mmoc/images/icons/iwarlocks.gif Warlock (Skills List (http://www.mmo-champion.com/index.php?page=680) / Talent + Glyph Calc. (http://talent.mmo-champion.com/?warlock))
Ritual of Summoning on a 2 minutes cooldown
Assuming nothing comes up at the last minute, this should be fixed tomorrow (Tue, Jan 27). (Source (http://blue.mmo-champion.com/28/14498471265-ritual-of-summoning-status.html))


Official Patch Notes
--------------------------------------------------------------------------------
- World of Warcraft Client Patch 3.0.8a
--------------------------------------------------------------------------------

We are updating World of Warcraft with a minor bug fix patch. For updated information regarding the changes made in this patch, please visit the official Bug Report forum at:
http://forums.worldofwarcraft.com/board.html?sid=1&forumId=10023 (http://forums.worldofwarcraft.com/board.html?sid=1&forumId=10023)

Bug Fixes
General
Human females Helms will no longer lift off the character's head during a Special2H animation or during a HoldRifle animation.

http://static.mmo-champion.com/mmoc/images/icons/ideathknights.gif Death Knight (Skills List (http://www.mmo-champion.com/index.php?page=650) / Talent + Glyph Calc. (http://talent.mmo-champion.com/?deathknight))
Howling Blast : The 5 second cooldown will now display correctly.

http://static.mmo-champion.com/mmoc/images/icons/ihunters.gif Hunter (Skills List (http://www.mmo-champion.com/index.php?page=620) / Talent + Glyph Calc. (http://talent.mmo-champion.com/?hunter))
All Hunter aspects are no longer on the global cooldown that is shared with other Hunter abilities. All hunter aspects are now on a separate 1 second cooldown.

http://static.mmo-champion.com/mmoc/images/icons/iwarlocks.gif Warlock (Skills List (http://www.mmo-champion.com/index.php?page=680) / Talent + Glyph Calc. (http://talent.mmo-champion.com/?warlock))
Ritual of Summoning no longer has a 2 minute cooldown and will now function as intended.

As for 3rd party patch mirrors:
Download the patch from the 3rd party site and place it in same folder as wow
run wow - it should download the downloader for the patch
the downloader will now verify the authenticity of the patch you downloaded
If the patch from the 3rd party site was valid, it will start patching, if not the blizzard downloader will try to fix it and download the broken pieces.

Edit:
And as for patches for Windows - check if the digital signature for the file is OK.
