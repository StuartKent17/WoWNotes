PvP Hotfixes - February 26
We've identified the following fixes that we're working to make live later today:

https://static.mmo-champion.com/mmoc/images/icons/imonks.gif Monk (Forums (http://www.mmo-champion.com/forums/341-Monk) / Skills (http://www.wowdb.com/spells/class-abilities/monk) / Talent Calculator (http://www.wowdb.com/talent-calculator#o) / Artifact Calculator (http://www.wowdb.com/artifact-calculator#p) / PvP Talent Calculator (http://www.wowdb.com/pvp-talent-calculator#o))

Brewmaster
Guard can no longer be applied to non-player targets such as pets and guardians.


https://static.mmo-champion.com/mmoc/images/icons/ipaladins.gif Paladin (Forums (http://www.mmo-champion.com/forums/273-Paladin) / Skills (http://www.wowdb.com/spells/class-abilities/paladin) / Talent Calculator (http://www.wowdb.com/talent-calculator#I) / Artifact Calculator (http://www.wowdb.com/artifact-calculator#JAAAAAAAA) / PvP Talent Calculator (http://www.wowdb.com/pvp-talent-calculator#I))
Protection Fist of Justice now reduces the cooldown of Hammer of Justice by 5 seconds (was 10 seconds).

https://static.mmo-champion.com/mmoc/images/icons/ipriests.gif Priest (Forums (http://www.mmo-champion.com/forums/274-Priest) / Skills (http://www.wowdb.com/spells/class-abilities/priest) / Talent Calculator (http://www.wowdb.com/talent-calculator#U) / Artifact Calculator (http://www.wowdb.com/artifact-calculator#VAAAAAAAA) / PvP Talent Calculator (http://www.wowdb.com/pvp-talent-calculator#U))
Discipline Fixed a bug that caused Atonement's heal to be higher than intended when using the honor talent Trinty.

https://static.mmo-champion.com/mmoc/images/icons/irogues.gif Rogue (Forums (http://www.mmo-champion.com/forums/275-Rogue) / Skills (http://www.wowdb.com/spells/class-abilities/rogue) / Talent Calculator (http://www.wowdb.com/talent-calculator#Q) / Artifact Calculator (http://www.wowdb.com/artifact-calculator#RAAAAAAAA) / PvP Talent Calculator (http://www.wowdb.com/pvp-talent-calculator#Q))
Assassination Fixed a bug that caused Kingsbane's damage to be higher than intended in PvP situations.

We expect to implement all of these changes simultaneously to all regions in a few hours, and you'll see them in today's full hotfixes update in our blog.
