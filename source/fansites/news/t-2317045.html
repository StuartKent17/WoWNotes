World of Warcraft BlizzCon Gear Store Merchandise
The Blizzard Gear store (https://gear.blizzard.com/us/blizzcon?product_list_limit=all&product_list_dir=asc) has been updated with new BlizzCon 2017 products. Orders are only open to Virtual Ticket holders today, but will open for everyone on 10/17. We've highlighted some of the more interesting items below, but you can see them all here (https://gear.blizzard.com/us/blizzcon?product_list_limit=all&product_list_dir=asc).


http://media.mmo-champion.com/images/news/2017/october/bconGear.jpg (https://gear.blizzard.com/us/blizzcon?product_list_limit=all&product_list_dir=asc)


World of Warcraft Sargeras Shirt
The shirt description (https://gear.blizzard.com/us/blizzcon/world-of-warcraft-sargeras-shirt-men-s) potentially explains the wound added to a map in Patch 7.3.2. Check out the shirt description for spoilers!


http://media.mmo-champion.com/images/news/2017/september/thumb/SilithusPhase01.jpg (http://www.mmo-champion.com/content/7000-Patch-7-3-2-PTR-Build-25079)

http://media.mmo-champion.com/images/news/2017/october/shirtSpoiler.jpg (http://media.mmo-champion.com/images/news/2017/october/shirt.jpg)


World of Warcraft Raptor Plushie
This raptor (https://gear.blizzard.com/us/blizzcon/world-of-warcraft-raptor-plushie) doesn't spoil anything in its description, but an interesting raptor mount was recently added in Patch 7.3.2. The ArmoredRaptor mount is still encrypted, so we don't know what it will look like. A new creature was also added that uses hoofprints and existing horse sounds in Patch 7.3.2, but the model is encrypted with the same key as the ArmoredRaptor. You can also pick up a Elekk plushie (https://gear.blizzard.com/us/blizzcon/world-of-warcraft-elekk-plushie)!


http://media.mmo-champion.com/images/news/2017/october/thumb/BlizzconRaptorPlushie.jpg (http://media.mmo-champion.com/images/news/2017/october/BlizzconRaptorPlushie.jpg)


Pirate Plush Pepe
This new Pepe (https://gear.blizzard.com/us/blizzcon/pirate-pepe-plush) features a pirate hate and eye patch, matching the Pirate Pepe (http://www.wowdb.com/objects/243280-pirate-pepe#model-viewer) you can see in game. He is part of the I Found Pepe! (http://www.wowdb.com/achievements/10053-i-found-pepe) achievement!


http://media.mmo-champion.com/images/news/2017/october/thumb/piratePepe2.jpg (http://media.mmo-champion.com/images/news/2017/october/piratePepe.jpg)


Kul Tiras Armor Sets
These are not for sale, but with BlizzCon approaching now is a good time to post a reminder about these sets that were added in Patch 7.3.


http://media.mmo-champion.com/images/news/2017/october/kultiras.jpg (http://www.mmo-champion.com/content/6784-Patch-7-3-PTR-Kul-Tiras-Armor-Sets-Updated-Rogue-Tier-21-Overwatch-Doomfist)


Overwatch - Halloween Terror Event
The Halloween Terror Event is live until Nov 1st (https://playoverwatch.com/en-us/events/halloween-terror/), bringing new cosmetics, a new game mode, and more! Check out the new intros and emotes on the official post.
