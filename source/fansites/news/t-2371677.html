Patch 7.3.5 Hotfixes - January 26
Achievements

Defeating Algalon (http://www.wowdb.com/npcs/32871-algalon-the-observer) should once again award the appropriate achievements.
Players can now correctly earn the achievement Souls of the Lost (http://www.wowdb.com/achievements/9026-souls-of-the-lost).


Classes

http://static.mmo-champion.com/mmoc/images/icons/irogues.gif Rogue (Forums (http://www.mmo-champion.com/forums/275-Rogue) / Skills (http://www.wowdb.com/spells/class-abilities/rogue) / Talent Calculator (http://www.wowdb.com/talent-calculator#Q) / Artifact Calculator (http://www.wowdb.com/artifact-calculator#RAAAAAAAA) / PvP Talent Calculator (http://www.wowdb.com/pvp-talent-calculator#Q))

Outlaw

Fixed a bug that caused Greenskin's Waterlogged Wristcuffs (http://www.wowdb.com/items/137099-greenskins-waterlogged-wristcuffs)'s to be removed when Pistol Shot (http://www.wowdb.com/spells/185763-pistol-shot) failed to cast.

Subtlety

Fixed a bug that prevented the gain of combo points after consuming the Regalia of the Dashing Scoundrel (http://www.wowdb.com/item-sets/1322-regalia-of-the-dashing-scoundrel) 4-piece set bonus.




Dungeons and Raids

Blackwing Lair (http://www.wowdb.com/zones/2677-blackwing-lair)

Nefarian (http://www.wowdb.com/npcs/11583-nefarian)

Addressed an issue where the Drakonid would fail to spawn as expected.


Ulduar (http://www.wowdb.com/zones/4273-ulduar)

Algalon the Observer (http://www.wowdb.com/npcs/32871-algalon-the-observer)

Made many tuning adjustments to Algalon (http://www.wowdb.com/npcs/32871-algalon-the-observer) to more closely match his difficulty in 7.3.2.


Temple of Ahn’Qiraj (http://www.wowdb.com/zones/3428-temple-of-ahnqiraj)

Silithid Royalty

Fixed a bug with the bugs despawning before players were able to loot them.




Items

Fixed a bug where some low level items were not able to be disenchanted or transmogrified.
Fixed a bug where incorrect textures were present on the Wristclasps of Righteous Reckoning (http://www.wowdb.com/items/127984-wristclasps-of-righteous-reckoning), Supreme Felchain Bracers (http://www.wowdb.com/items/127983-supreme-felchain-bracers),Bracers of Perfect Discomfort (http://www.wowdb.com/items/127981-bracers-of-perfect-discomfort), and Terrorweave Wristwraps (http://www.wowdb.com/items/127982-terrorweave-wristwraps). This fix should provide the correct texture and restore set completion to some players.


Player versus Player

Character item level is now set to 950 during Tournament Rules Wargames.
Concordance of the Legionfall (http://www.wowdb.com/spells/239042-concordance-of-the-legionfall?artifactRank=1) is now set to rank 20 during Tournament Rules Wargames.
http://static.mmo-champion.com/mmoc/images/icons/imonks.gif Monk (Forums (http://www.mmo-champion.com/forums/341-Monk) / Skills (http://www.wowdb.com/spells/class-abilities/monk) / Talent Calculator (http://www.wowdb.com/talent-calculator#o) / Artifact Calculator (http://www.wowdb.com/artifact-calculator#p) / PvP Talent Calculator (http://www.wowdb.com/pvp-talent-calculator#o))

Mistweaver

The healing of Way of the Crane (http://www.wowdb.com/spells/216113-way-of-the-crane) is properly decreased by healing reduction effects such as Dampening (http://www.wowdb.com/spells/110310-dampening) or Mortal Strike (http://www.wowdb.com/spells/12294-mortal-strike).




Quests

Neptulon's Cache (http://www.wowdb.com/objects/205216-neptulons-cache) now drops the Remnant of Water (http://www.wowdb.com/items/138953-remnant-of-water) for quest “Controlling the Elements (http://www.wowdb.com/quests/42971-controlling-the-elements)”.
The Duskwood (http://www.wowdb.com/zones/10-duskwood) quest "Cry For the Moon (http://www.wowdb.com/quests/26760-cry-for-the-moon)" should now appear on the map if it is the next quest in the zone's storyline.



Blue Posts
I just logged in, and to my surprise, there is a level 90 boost on my character select screen...What is it from? Should I be worried about some sort of account compromise? I have the authenticator and everything. A few weeks ago I purchased a level 100 boost for a warlock. I looked through my transaction history, and there aren't any random purchases. Is this a bug? A random gift from Blizz?

It looks like we're running a process that is switching the level 90 boosts that used to be bound only to the WoW license to make them Blizzard Battle.net wide. So some of you likely had a level 90 boost from a secondary WoW license that is now available for use on any of your WoW licenses on that account. For those currently missing boosts, the process is still running, they should show back up. Once the process completed we'll see how everything looks.
