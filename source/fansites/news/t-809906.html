Cataclysm Raid Progression Update
A couple of guilds are leading the world progression, almost all the normal modes are down now and only Nefarian is left for 4 of those guilds. World of Warcraft Cataclysm Performance
Tom's Hardware (http://www.tomshardware.com/) posted a very useful performance guide for Cataclysm. The entire article is a must-read (http://www.tomshardware.com/reviews/world-of-warcraft-cataclysm-directx-11-performance,2793.html) if you plan to upgrade your computer anytime soon, keep in mind that this article is only about performance in WoW Cataclysm and things can be very different in other games (mostly thinking about multi GPU cards here).

There's also a few comparisons of DirectX9 vs DirectX11 performance, if you have an high-end GPU card it looks like you will definitely want to activate the DX11 Support to get even more FPS! To do that, simply edit the config.wtf file in your /WTF/ folder and add SET gxApi "d3d11" to it.

Regarding Dx11:
DX11 mode is available to users with DX10 & DX11 cards, running Windows 7, or a fully up to date Vista install (requires SP2 and a specific update afterwards). Wow will internally switch pixelshaders, etc based on what type of card is installed (Dx10 cards will run 4.0, DX11 cards will run 5.0).

Also note those running Windows 7 with a DX11 card shouldn't need to enable it, as the engine SHOULD be automatically setting you to this (whether it is or not is another story).

Enabling DX11 mode on an XP system will have the game engine reset the setting to DX9.

Devs are still learning about optimizing for DX11, and are actively working with Microsoft to optimize the game engine further for those able to use DX11 mode.

All of this is from what Blizz reps have stated time & time again in the US Wow Tech Support forum.
