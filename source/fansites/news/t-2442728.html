Wait, how can you hide Activision games from the launcher? Is there a setting for it?

No setting but there was a way of modifying the BDA so that entire tab is hidden.

It was placing a QML file in a special folder that has the visible property for that tab set to hidden. Unfortunately with how the battle.net launcher is designed , every time they push a new build you have to re-apply the mod.

Not because it was being caught by some integrity check or anything. It's because of how the battle.net client is installed - each build lives inside it's own folder with most if not all it's resources and things inside it (aka Battle.net.xxxx where xxxx is a build number). Each time the battle.net team push out a new build of the BDA the battle.net desktop app / client will load the new build ..... and the modified file is in the old build folder.

EDIT : Looks like they plugged that way of modifying the BDA. Just tested it myself.
