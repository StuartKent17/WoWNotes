Legion Collector's Edition Mount - Illidari Felstalker
The Illidari Felstalker (http://www.wowdb.com/spells/189998-illidari-felstalker) is the Legion Collector's Edition Mount. Keep in mind that while it is flagged as a flying mount, there are currently no specific flying animations, so it uses swimming animations. There is no official confirmation that it will be able to fly.

Patch 6.2.2 - Darkmoon Faire Updates
The Darkmoon Seesaw (http://www.wowdb.com/items/97994-darkmoon-seesaw) can be purchased from Gelvas Grimegate (http://www.wowdb.com/npcs/14828-gelvas-grimegate) for 50 x Darkmoon Prize Ticket (http://www.wowdb.com/currencies/515-darkmoon-prize-ticket). You can jump on while Lunk is still riding to get launched off and the Childlike Wonder (http://www.wowdb.com/spells/143034-childlike-wonder) buff, which shrinks you by 33%.

Don't forget to take a look at the new fishing related things (http://www.mmo-champion.com/content/5000-Patch-6-2-Darkmoon-Faire-Mythic-HFC-Mythic-Gorefiend-Video-Blue-Tweets-DLC-496) that were added in Patch 6.2 if you haven't already.

Be sure to pick up WHEE! (http://www.wowdb.com/spells/46668-whee) from the Darkmoon Carousel or the buff from Darkmoon Top Hat (http://www.wowdb.com/items/93730-darkmoon-top-hat) for extra reputation if you are still doing the Tanaan Daily quests!
