i wouldn't say its too complicated to explain....

The CD's are not containing everything which makes world of warcraft work.

In modern software development- especially if you build for the masses - you have a server / client system.

That is : all home PCs connect to one server (your realm) and the server cares for syncing and distributing the right information at the right time to the other people playing beside you.

Now the code for the server is NOT on the CD's.
if it was it would be super easy to setup your own server and then dont pay for the subscription.

Lets put that aside however.

Now that we straightend this out we come to the second part:

The big problem with setting up the server for classic is that the old code from back in the days does not run on newer computers/servers it basically only runs on the old server hardware which was partially up for sale on ebay. It was completly not available to blizzard anymore.
Only by buying such a server from the new owner they were able to manage to get it back up and running.

They could have released that - but that would include getting back all the servers they previously sold (or many of them)
additionally they have listed concerns that the old server software has its downsides - its not technologically as advanced as the servers are today. so you get back all the troubles they had in the past with the servers.

Thats when they decided to completly rewrite the server side to use the current technology and tailor classic on top of it.

And thats basically a lot of work :)
