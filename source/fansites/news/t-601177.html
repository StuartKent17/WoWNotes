Microsoft released yesterday a hotfix for Windows 98, 2000, 2003 , XP , and Vista.
http://www.microsoft.com/technet/security/bulletin/MS07-017.mspx
This was released to fix the "Windows Animated Cursor Remote Code Execution Vulnerability".

This vulnerability has to be fixed for WoW players because it is used to install a new version of an already well-known password stealer. The original one was released during October 2006, but the new version isn't detected by all antiviruses yet. If you want more informations about the old password stealer check :
http://www.f-secure.com/v-descs/wow.shtml

If you want more informations about this vulnerability you can also check the following links :
Post on the official forums (http://forums.wow-europe.com/thread.html?topicId=102690401&sid=1) about how to keep your computer safe.
29/03 report from SANS Internet Storm Center (http://isc.sans.org/diary.html?date=2007-03-29)
30/03 vulnerability report for Microsoft E-Mail clients (http://isc.sans.org/diary.html?date=2007-03-30)
Video on McAfee blog (http://www.avertlabs.com/research/blog/?p=233) showing how annoying this exploit can be, even when it's not used to steal passwords
