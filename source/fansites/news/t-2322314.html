Hallow's End 2017 Live
Hallow's End (http://www.wowdb.com/world-events/324-hallows-end) is here once again through Nov 1st and this year brings a few updates:


Naxxy (http://www.wowdb.com/npcs/117341-naxxy) - This mini Naxxramas pet can be purchased for 150 x Tricky Treats (http://www.wowdb.com/items/33226-tricky-treat)
Horse Head Costume (http://www.wowdb.com/items/151271-horse-head-costume) and Horse Tail Costume (http://www.wowdb.com/items/151270-horse-tail-costume) - Each half of this costume can be purchased for 150 x Tricky Treats (http://www.wowdb.com/items/33226-tricky-treat)
Exquisite Costume Set: "Xavius" (http://www.wowdb.com/items/151268-exquisite-costume-set-xavius) - This costume can be purchased for 200 x Tricky Treats (http://www.wowdb.com/items/33226-tricky-treat)
Hallow's End Cosmetic items can be transmogged during the event
The event boss loot is now scaled to item level 880

Beta Build Version Number Update
This afternoon the beta version of World of Warcraft was updated to Patch 8.0.0 - Build 25338. There is no data or actual patch, this is just a version number that is mentioned on the patch server. It is likely related to getting a build of the game ready for BlizzCon.
