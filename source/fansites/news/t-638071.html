Tier 8 Sets - Final Model Preview
Tonight, we saw the first preview of the stats of a Tier 8 set on the PTRs, for the moment they're only T7.25 sets with an higher ilvl (and armor).


http://static.mmo-champion.com/mmoc/images/news/2009/march/tier8_warrior_10_tank_small.jpg (http://static.mmo-champion.com/mmoc/images/news/2009/march/tier8_warrior_10_tank.jpg)

I guess it's a good time to post the final preview of the Tier 8 models, there are actually plenty of changes since the last time I posted screenshots of tier 8. The biggest change is probably that all the "old" tier 8 sets have been upgraded with high-resolution textures. Here are the changes for each class since the last time I posted these sets:

http://static.mmo-champion.com/mmoc/images/icons/ideathknights.gif Death Knight - Higher resolution textures.
http://static.mmo-champion.com/mmoc/images/icons/idruids.gif Druid - All races added, robe version of the set added.
http://static.mmo-champion.com/mmoc/images/icons/ihunters.gif Hunter - Higher resolution textures.
http://static.mmo-champion.com/mmoc/images/icons/images.gif Mage - Higher resolution textures, and finally decided that the pants version of the set might be the final version for players. As of today there isn't any texture for Tier 8 Mage robe in the game files.
http://static.mmo-champion.com/mmoc/images/icons/ipaladins.gif Paladin - All races added, robe version of the set added.
http://static.mmo-champion.com/mmoc/images/icons/ipriests.gif Priest - Higher resolution textures.
http://static.mmo-champion.com/mmoc/images/icons/irogues.gif Rogue - Higher resolution textures. There are still a few bugs with this set and the green recolor isn't available on male characters, undead females are also unavailable.
http://static.mmo-champion.com/mmoc/images/icons/ishamans.gif Shaman - All races added, robe version of the set added.
http://static.mmo-champion.com/mmoc/images/icons/iwarlocks.gif Warlock - Higher resolution textures.
