Update Patch 4.1 PTR - Official Notes Update
The Patch 4.1 PTR Notes (http://blue.mmo-champion.com/topic/160430/patch-41-public-test-realm-notes-updated-march-1) have been updated once again with the 2 missing changes from build 13698.
http://static.mmo-champion.com/mmoc/images/icons/iwarriors.gif Warrior (Forums (http://www.mmo-champion.com/forums/278-Warrior) / Cataclysm Talent Calculator (http://www.wowtal.com/#k=..warrior) / Skills/Talents (http://www.mmo-champion.com/content/1676-Warrior-Cataclysm))
Arms

Strikes of Opportunity damage has been increased by 10% to keep stat parity with other strike changes.


Fury

Unshackled Fury now only grants 2 base points of mastery, down from 8. This change offsets the Flurry boost to better control Fury burst damage.



Patch 4.1 on PTRs - Build 13698
A new build will be deployed on test realms today. Most of these changes were already documented in the latest PTR notes update (http://www.mmo-champion.com/content/2193-Patch-4.1-Notes-February-28-Update) but keep an eye out for new things!

Arms
