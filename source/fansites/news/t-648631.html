Changes to Items with Stealth Detection
We recently made some changes to items that grant stealth detection. We felt these items provided a necessary advantage in competitive PVP and forced players to use lower level equipment even when more powerful items were available to them.

This item can no longer be used in the Arena:
Ultra-Spectropic Detection Goggles (http://www.wowhead.com/?item=23762)

We have removed stealth detection from the following items:
All epic engineering goggles

We have reduced the stealth detection of the following items:
Catseye Ultra Goggles (http://www.wowhead.com/?item=10501)
Blood Guard's Dragonhide Gauntlets (http://www.wowhead.com/?item=16496)
Marshal's Dragonhide Gauntlets (http://www.wowhead.com/?item=16448)
Knight-Lieutenant's Dragonhide Gloves (http://www.wowhead.com/?item=16397)
General's Dragonhide Gloves (http://www.wowhead.com/?item=16555)
Bloodvine Lens (http://www.wowhead.com/?item=19998)
Knight-Lieutenant's Dragonhide Grips (http://www.wowhead.com/?item=23280)
Blood Guard's Dragonhide Grips (http://www.wowhead.com/?item=22863)
The Night Watchman (http://www.wowhead.com/?item=31333)

All of these changes are already active and the tooltips will be updated with the next minor patch.
