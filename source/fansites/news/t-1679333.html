Starfall (http://ptr.wowdb.com/spells/48505) effect is now only suppressed when shapeshifting into a non-Moonkin Form (http://ptr.wowdb.com/spells/24858). Shifting back into the appropriate form will allow the effect to resume.

Armor Sets

Tier-17 4-piece set bonus for Brewmaster Monks now only triggers when you Purify (http://ptr.wowdb.com/spells/527) Moderate or Heavy Stagger.
PvP 2-piece set bonus for Brewmaster and Windwalker Monks now only triggers while in PvP combat.
PvP 4-piece set bonus for Brewmaster Monks now confers 75% of the Guard (http://ptr.wowdb.com/spells/115295) effect to the 2 nearest allies.


http://static.mmo-champion.com/mmoc/images/icons/irogues.gif Rogue (Forums (http://www.mmo-champion.com/forums/275-Rogue) / Skills (http://www.wowdb.com/spells/class-abilities/rogue) / Talent Calculator (http://www.wowdb.com/talent-calculator#Q))

Combat

Blade Flurry (http://ptr.wowdb.com/spells/13877) is now able to trigger from Multistrikes.


Raids, Dungeons, and Scenarios

Highmaul

Kargath Bladefist: Resolved an issue where players can incorrectly get locked to the instance after wiping to Kargath on Mythic difficulty.
Imperator Mar'gok: Gorian Warmage's Fixate ability should now only cause the targeted player to drop out of stealth instead of incorrectly causing all stealthed players in the raid to drop out of stealth.


Legacy

Blackrock Caverns

Rom'ogg Bonecrusher's Skullcracker ability should now correctly deal damage to Angered Earth elementals, allowing the achievement Crushing Bones and Cracking Skulls to be completed.


PvP

Alliance and Horde Conquest vendors should now offer non-set cloth gloves and non-set plate chest pieces for sale.
Ashran

Amphitheater of Annihilation: Mandragoraster's Fertile Spit ability now caps out at 6 nearby targets.


Battlegrounds and Arenas

Arathi Basin: Players should now always have a fixed 25-second resurrection timer while in a Rated Battleground match.
Deepwind Gorge: Using Ice Floes or Aspect of the Fox should no longer incorrectly allow players to capture a point while moving.
