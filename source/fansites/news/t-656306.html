These tests are designed to sort out bugs and determine tuning. If I were Blizzard, I'd deliberately under-tune PTR fights first time around, just to make sure people killed it so the last phase (if any) gets tested.

Furthermore, this is easy mode. If you're raiding Ulduar25* hard modes, this fight is probably supposed to be somewhat easy for you: you're supposed to WTFPWN normal Trial of the Crusader in short order so you can do Trial of the Grand Crusader.
